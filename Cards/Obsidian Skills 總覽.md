[Obsidian Skills](https://github.com/kepano/obsidian-skills) 是一組讓 Claude Code 操作 Obsidian vault 的技能。

| Skill | 用途 |
|-------|------|
| `obsidian:obsidian-cli` | 讀取、建立、搜尋筆記，管理 tasks 與 properties |
| `obsidian:obsidian-markdown` | 撰寫 Obsidian Flavored Markdown（wikilinks、callouts、embeds） |
| `obsidian:json-canvas` | 建立與編輯 `.canvas` 畫布檔案 |
| `obsidian:obsidian-bases` | 建立與編輯 `.base` 資料庫視圖 |
| `obsidian:defuddle` | 從網頁擷取乾淨 Markdown（本機未安裝，用 WebFetch 取代） |
