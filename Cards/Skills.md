# Claude Code Obsidian Skills

## obsidian:obsidian-cli

與 Obsidian vault 互動的主要工具。

**功能**
- 讀取、建立、搜尋筆記
- 管理 tasks、properties
- 開發與除錯 plugin（reload、截圖、DOM 檢查）

## obsidian:obsidian-markdown

撰寫與編輯 Obsidian Flavored Markdown。

**功能**
- Wikilinks `[[筆記名稱]]`
- Callouts `> [!tip]`
- Frontmatter properties
- Embeds、標籤、雙向連結

## obsidian:json-canvas

建立與編輯 `.canvas` 畫布檔案。

**功能**
- 新增 nodes、edges、groups
- 視覺化連結筆記、心智圖、流程圖

## obsidian:obsidian-bases

建立與編輯 `.base` 資料庫視圖檔案。

**功能**
- Table view、Card view
- 篩選、排序、公式
- 統整多張筆記的 properties

## obsidian:defuddle

從網頁擷取乾淨的 Markdown 內容。

**功能**
- 移除導覽列、廣告等雜訊
- 輸出精簡的 Markdown，節省 token

> [!warning] 本機未安裝 Defuddle CLI，直接用 WebFetch 取代
